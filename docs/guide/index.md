# 介绍

yr-ui 是一款优秀的微信小程序 UI 组件库，提供了丰富的组件，帮助开发者快速构建高质量的微信小程序。

## 特性

- 🎨 **精美设计** - 遵循微信小程序设计规范，提供一致的视觉体验
- 📦 **丰富组件** - 提供 130+ 个高质量组件，覆盖各种业务场景
- 🔧 **灵活配置** - 支持全局配置和组件级别的属性定制
- 📱 **完美适配** - 适配各种屏幕尺寸，提供良好的移动端体验
- 🚀 **持续更新** - 持续维护更新，及时修复问题
