# Spin 旋转动画

为内容添加无限旋转动画。

## 基础使用

```html
<yr-ui-spin>
  <view class="loading-icon"></view>
</yr-ui-spin>
```

## API

### Props

| 参数 | 说明 | 类型 | 默认值 | 可选值 |
| --- | --- | --- | --- | --- |
| width | 宽度 | number/string | 0 (auto) | - |
| height | 高度 | number/string | 0 (auto) | - |

### Slots

| 插槽名 | 说明 |
| --- | --- |
| default | 需要旋转的元素 |
